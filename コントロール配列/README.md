# 概要

VBAでコントロール配列を実現する方法です

VBやC#ならコントロール配列を作れますが、VBAは作れません

しかし、クラスを使うことによりVBAでもコントロール配列っぽいものを作れます


# 説明

3つのパターンを作りました

1. コントロールの個数固定、ユーザーフォーム使用
1. コントロールの個数固定、ユーザーフォーム未使用
1. コントロールの個数未定、ユーザーフォーム使用
